# avl-tree-structure_Roman_Soboliev_ki_2026
Доброго ранку/дня/вечора!

Наведу стислий технічний огляд програми.

Програма являє собою консольний застосунок, який реалізує структуру даних збалансованого двійкового дерева пошуку - АВЛ-дерева, написану мовою C++. Головна мета програми полягає в демонстрації алгоритмів автоматичного балансування структури при додаванні нових елементів, що гарантує виконання операцій за логарифмічний час.

Архітектура побудована за принципами об'єктно-орієнтованого програмування та розділена на заголовковий файл з описом класів і файл реалізації. Клас дерева керує вузлами, кожен з яких зберігає цілочисельний ключ, поточну висоту та вказівники на нащадків. Балансування дерева забезпечується методами обчислення різниці висот піддерев та виконанням відповідних одинарних або подвійних поворотів (лівого чи правого) у випадках виникнення перекосу. Також реалізовано безпечне керування динамічною пам'яттю за допомогою деструктора, який рекурсивно очищає всі створені вузли при завершенні роботи.

Користувацький інтерфейс реалізовано у вигляді текстового меню. Користувач має можливість додавати вузли по одному вручну, завантажити тестовий масив з десяти наперед визначених чисел для швидкої перевірки алгоритму, а також вивести поточну ієрархію дерева на екран у наочному вигляді з відступами. Процес ручного введення даних захищено базовою валідацією, яка перевіряє стан потоку та відхиляє спроби введення символів або дробових чисел, зберігаючи стабільність роботи основного циклу програми.
